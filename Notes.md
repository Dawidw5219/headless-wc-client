# Headless WooCommerce Client

Client for Headless WooCommerce Wordpress Plugin

## Dev env

1. Run
   yarn dev

## Local Setup

1. Run inside this project folder
   npm link

2. To use this package inside diffrents projects localy you use
   npm link headless-wc-client
